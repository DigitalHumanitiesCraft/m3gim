Datenschutz

Diese Website verwendet keine Cookies, kein Tracking und keine Analytics-Dienste. Es werden keine personenbezogenen Daten erhoben, gespeichert oder an Dritte weitergegeben. Die Anwendung läuft vollständig im Browser; alle Daten werden beim Seitenaufruf von GitHub Pages geladen.

Der Wissenskorb speichert gesammelte Objekte ausschließlich im Session Storage des Browsers. Diese Daten werden beim Schließen des Browser-Tabs automatisch gelöscht und verlassen zu keinem Zeitpunkt das Gerät.

Technologie

Frontend: Vanilla JavaScript (ES6-Module), D3.js v7. Pipeline: Python 3.11+. Datenmodell: JSON-LD (RiC-O 1.1 + m3gim-Erweiterung). Hosting: GitHub Pages. Typografie: Inter, Source Serif 4, JetBrains Mono.
